## Features

- **Minimalist Design**: A clean, distraction-free interface with a premium cyberpunk aesthetic.
- **Dynamic Themes**: Switch between various themes including Neon Cyberpunk, Glass Nature, and Midnight Void.
- **Performance Tracking**: Real-time statistics for words per minute (WPM), accuracy, and error tracking.
- **Visual Feedback**: Dynamic glow effects and glassmorphism elements enhance the user experience.
- **Offline Capable**: Works without an internet connection after initial load.

## Getting Started

1. Clone the repository.
2. Install dependencies with `npm install`.
3. Start the development server with `npm run dev`.
4. Open your browser and navigate to `http://localhost:3000`.

## Technology

Built with Next.js, TypeScript, and SCSS.
